总结

一个人在这边上班，刚开始一两个月生活比较规律，锻炼身体学习进步，后面生活就乱了，很多时间在玩手机，即使手机上也没啥真正有用的东西，浪费了大量时间，还经常熬夜。手机是信息工具，应该有目标的看，而不是漫无目的，有害无益，全是娱乐信息。
对社会认识更深刻，努力赚到钱还房贷，将房贷压力干掉。
争取能赚到人生第一桶金，之后做自己感兴趣的事情，或者找到自己非常感兴趣的事情，有机会do it。
培养事业上的朋友。
培养自己的专业能力，专注其中，只争朝夕。
